## Featured Work

### 🔬 Computer Vision Inference Pipeline
> Production AI system · 40K+ images/day · deployed

Batched inference architecture with async preprocessing, GPU memory pooling (fp16), and Redis job queuing. Built to handle variable load with horizontal scaling and automatic retry on failure.

`PyTorch` `Python` `Redis` `Docker` `Node.js` `AWS S3`

```
Ingestion → Redis Queue → GPU Workers (batch=32, fp16) → MongoDB + S3 → Webhook
```

### 📄 NLP Resume Parser
> Production · 94% extraction accuracy · <1.2s parse time

Extracts structured data from unstructured resumes across PDF and DOCX formats, multiple layouts, noisy text. Used in a live candidate screening workflow.

`Python` `spaCy` `Transformers` `FastAPI` `Docker`
